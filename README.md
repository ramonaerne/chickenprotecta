# chickenprotecta
IoT Chicken Coop Door System

### Adapting the code
```
cp chickenprotecta-public.ino chickenprotecta.ino
```
and fill out auth token and wifi ssid/password

### Pushing to the repo
To upload any changes while not uploading ssid/password and auth token use
```
./add.sh
```
instead of `git add chickenprotecta.ino`
